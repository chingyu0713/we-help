關於這份申請網頁，分享一個開發時的技術心得。

在開發這個網頁的過程中，我遇到了一個挑戰：如何實現一個符合現代 RWD（響應式網頁設計）規範的 Navbar。加入 Navbar 後，需要對排版進行大幅調整，以確保在不同裝置上提供流暢的用戶體驗。起初，我手動添加空隙，對靜態內容還能運行，但在點擊連結時，內容會意外移位。為了解決這個問題，我使用了 scroll-padding-top，確保錨點連結具有一致的間距。同時，我還加入了 CSS 變量 :root { --navbar-height: }，以統一 Navbar 的高度設定。這個方法不僅解決了問題，還提升了設計的可維護性和擴展性，為未來的更新奠定了基礎。
